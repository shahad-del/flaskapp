Python classes let you share behavior (a.k.a. methods) and state (a.k.a. attributes).

Creating a new object from a class looks very like a function call. Remember: to create an object called mycount from a class called CountFromBy, you’d use this line of code: 

mycount = CountFromBy()
Here mycount is an object that is created from countFromBy class

When an object is created from a class, the object shares the class’s code with every other object created from the class. However, each object maintains its own copy of the attributes. 

mycount = CountFromBy() 
    
object2 = countFromBy()

Both objects mycount and object2 shares all those methods in the class countFromBy but have their own attributes.That means: 
mycount = CountFromBy('Number')
object2 = CountFromBy('nikhil')
This makes both the object different.

You add behaviors to a class by creating methods. A method is a function defined within a class.To add an attribute to a class create a variable.
